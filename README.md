# 2026 Republic of Korea - Cheong Wa Dae Backup Project / 2026 대한민국 청와대 백업 프로젝트

## Overview / 개요
This project is a comprehensive digital archive of the official website of the Blue House (Cheong Wa Dae) of the Republic of Korea, captured in 2026. It aims to preserve historical digital records, including schedules, speeches, organization charts, and media content.

이 프로젝트는 2026년 기준 대한민국 청와대 공식 홈페이지의 디지털 기록을 보존하기 위한 종합 아카이브입니다. 대통령 일정, 연설문, 조직도 및 미디어 콘텐츠를 포함한 역사적 디지털 기록의 원형 보존을 목적으로 합니다.

## Usage / 사용법
To view the archived website, open `index.html` in any modern web browser. All static resources are stored locally to ensure long-term accessibility without external dependencies.

아카이브된 웹사이트를 열람하려면 웹 브라우저에서 `index.html` 파일을 실행하십시오. 모든 정적 리소스는 로컬에 저장되어 있어 외부 연결 없이도 장기적인 접근성이 보장됩니다.
